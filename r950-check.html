1
MFA + Conditional Access Enforcement
Why: No MFA = hackers bypass password in seconds using leaked creds. What we check: Per-user MFA, CA policies, legacy auth blocked, risky sign-ins. Screenshot of Entra ID logs. Gauteng reality: Only 2 of 47 law firms had CA enforced. Risk if ignored: Full mailbox takeover, wire fraud.
CRITICALFix time 15min, included free
2
Secure Score & Global Admin Hygiene
Why: Secure Score under 40% = low-hanging fruit. Too many Global Admins = blast radius. Check: Secure Score export, Global Admins list, stale admins. Common SA fail: Ex-IT guy still Global Admin 2 years later. Risk: Insider threat + POPIA failure.
3
Mailbox Forwarding, Delegates & Data Exfil
Why: Silent forwarding = ex-staff reading tenders. Check: Transport rules, SMTP forwarding, delegates, audit log 90 days. Real case: Centurion estate agency leaking offers via rule. Risk: POPIA + FICA non-compliance.
4
SharePoint / Teams Sharing Links & Guest Access
Why: “Anyone with link” = client ID docs indexable. Check: Sharing controls, anonymous expiry, guest access report. Common SA: Law firms sharing contracts with “Anyone” never expiring. Risk: POPIA Section 26.
5
DKIM, SPF, DMARC & Anti-Phishing
Why: Without these, anyone spoofs your @company.co.za. Check: DNS TXT, DMARC policy, Defender policies. SA: Banks block you if DMARC missing. Risk: Invoice fraud.

Category 2: Backup, Recovery & POPIA
Backup that hasn’t been restored is just a hope. POPIA requires proof.
5 CHECKS
6
M365 Backup vs Sync Reality Check
Why: Microsoft uptime ≠ your data protection. Sync deletes cloud if ransomware. Check: Real backup exists? Versioning 30+ days? Test restore 3 files live. SA stat: 34/47 no backup, just sync. Risk: Total loss + POPIA Section 20.
TOP FAILURE
7
Server / NAS Image & 3-2-1 Compliance
Why: File backup ≠ server restore in 1hr. Check: Bare-metal schedule, offsite SA copy, encryption, 30-day logs. NAS immutable snapshot on? Typical Pretoria: NAS same room as server, fire = both gone. Risk: R150k+ downtime.
8
RTO / RPO Reality & Restore Drill Log
Why: POPIA asks how fast you restore, not if you backup. Check: Documented RTO/RPO, last test date, who knows restore, key offline. We attempt SharePoint restore on-site. Fail: No one knows password.
9
POPIA Retention & Access Log Gap
Why: POPIA requires minimal retention + audit trail. Check: Retention policy? HR IDs from 2015 on share? M365 auditing on? Bulk download alerts? SA SME: Keeping CVs, IDs 7+ years open. Risk: Fine up to R10m.
10
Encryption - BitLocker / DLP / Device
Why: Stolen unencrypted laptop = mandatory breach notification. Encrypted = safe harbor. Check: BitLocker enforced? Keys in Entra? USB blocking? DLP for SA IDs. Common: Only MD’s laptop encrypted. Risk: Single theft = 100 notifications.

Category 3: Network, Firewall & WiFi
Your internet box is your castle gate. Most gates are wide open.
5 CHECKS
11
Firewall Firmware, IPS & Default Creds
Why: 2019 firmware = known exploits, botnets scan Gauteng daily. Check: Model, firmware age, IPS on?, default admin, remote mgmt exposed. Photo of rack. Real: Mikrotik admin blank in Montana. Risk: Full takeover.
12
Open Ports & RDP / VPN Exposure
Why: RDP open = #1 ransomware entry in SA. Check: nmap public IP, uPnP, port forwarding, VPN type. Shodan check. Midrand: Sage RDP 3389 weak password. Risk: Ransomware in 2hrs.
13
WiFi Security - WPA3, Guest Isolation
Why: Staff phone same WiFi as payroll = lateral move. Check: WPA2/WPA3, PSK strength, guest VLAN, isolation, AP firmware. Common: Guest WiFi = company name 123. Risk: Reception can browse shares.
14
DNS, DHCP & Internet Resilience
Why: Slow internet often DNS poison. Check: DNS health, DHCP conflicts, dual WAN, 4G backup SIM, DNS filtering. SA: Load-shedding kills fibre box, no failover. Risk: Payroll day offline.
15
IoT, Printers & Forgot Devices
Why: Printer default admin = stores every ID copy. Check: Printer/cCTV default creds, firmware, SMBv1, scan-to-mail creds. Example: Centurion medical scan-to-email using MD password plaintext. Risk: Data leakage.
